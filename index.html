IANA SYSTEMS - AI-Human Hybrid Systems for security and analytics.

Unique security system - Analytics - We build tools that matter most

IANA SYSTEMS is a toolkit for building web applications and libraries with Rust and Wasm, composed of modular crates. IANA SYSTEMS crates include example code, both in their respective example folders, and commented in their code, as well as API documentation.

Using IANA SYSTEMS

IANA SYSTEMS is a modular toolkit: Each of its crates can either be used via the umbrella gloo crate, which re-exports all of them for a stream-lined, holistic experience, or each crate can be used independently.
